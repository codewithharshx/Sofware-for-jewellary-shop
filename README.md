## ğŸš€ How to Start and Run

### 1. Pre-requisites
Ensure you have [Node.js](https://nodejs.org/) installed on your computer.

### 2. Installation
Open your terminal (PowerShell or CMD) in the project root folder and run:
```bash
npm run install:all
```
This will install all necessary dependencies for both the backend and frontend.

### 3. Running the Project
The project requires both the server and the user interface to be running at the same time. You will need to open **two** terminal windows.

#### Terminal 1: Start the Backend (Server)
```bash
npm run start:backend
```
The server will start on `http://localhost:5000`.

#### Terminal 2: Start the Frontend (Vite)
```bash
npm run start:frontend
```
The frontend will start on `http://localhost:5173`. Open this URL in your web browser.
